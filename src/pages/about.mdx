The Field Guide for WooCommerce is an open, community-maintained operational playbook — the book that should ship with WooCommerce but doesn't.

It covers what you actually run into when you run a store: every product type and how it handles inventory; the invisible layer beneath payments; the specifics of tax in the US and the EU; the protocols for when something breaks; and the code-level patterns for when you outgrow extensions.

## Who it's for

Three audiences, same pages. The lede of each section is plain English — written so a merchant new to WooCommerce can follow along. Deeper mechanics and code appear under *For developers* later in the same section, so agency builders and developers get what they need too.

- **Merchants** running their own Woo store — especially when you've hit a wall and the official docs don't answer the question.
- **Agencies and builders** shipping client sites — pre-launch checklists, staging pitfalls, handoff practices.
- **Developers** extending WooCommerce with custom code — hook patterns, REST API, HPOS-safe queries.

## How to read it

Browse the full index, or start with the first section of each part (see the homepage). Every section stands alone — you don't have to read in order. Search covers the whole book.

## For agencies handing off to clients

The whole guide is licensed under [Creative Commons Attribution 4.0](https://creativecommons.org/licenses/by/4.0/). That means you can fork it, rewrite sections in your own voice, strip it down to the ten things your client actually needs, and hand it off — as long as you credit the source.

The whole book is plain Markdown in a public [GitHub repo](https://github.com/namiokuzono/fieldguide-for-woocommerce). Clone it, edit in your editor of choice, build a client-branded version. Every page also has a "Print" button for a quick PDF handoff.

## How it's maintained

It's a living document. Open an issue or a pull request on GitHub — corrections, new sections, better examples are all welcome. See the [contributing guide](https://github.com/namiokuzono/fieldguide-for-woocommerce/blob/main/CONTRIBUTING.md) for the style.
